# Contribuir a TPV Ultra Smart

## Reglas
1. No modificar `app.py` sin tests
2. Probar cambios con `python3 tests/`
3. No subir APK al repositorio
4. Usar ramas feature/: `feature/nueva-funcion`

## Estructura
- `app/src/main/python/` → Backend
- `app/src/main/assets/` → Frontend
- `tests/` → Tests unitarios

## Flujo
1. Fork del repositorio
2. Crear rama feature/
3. Hacer cambios + tests
4. Pull Request a main
